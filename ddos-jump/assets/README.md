### Как заменить “модельки” в игре

В игре поддерживаются спрайты (PNG) для:
- герой: `player.png`
- герой во время выстрела: `player_shoot.png` (опционально)
- платформ: `platform.png`
- врагов (5 разных): `enemy0.png` … `enemy4.png` (опционально)
- пуль: `bullet.png`

Положи файлы в эту папку `assets/`. Имена должны совпадать с указанными выше.

Если размеры у спрайтов другие, их отмасштабирует Canvas (через `drawImage` до размеров хитбоксов игры).
