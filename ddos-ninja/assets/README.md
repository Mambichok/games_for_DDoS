# Assets for DDoS-Ninja

Place your PNG files in this folder with exact names:

- `logo.png` - logo in top-left corner
- `heart.png` - life icon in top-right corner (3 hearts)
- `forbidden.png` - special item that must NOT be sliced
- `fruit-1.png`
- `fruit-2.png`
- `fruit-3.png`
- `fruit-4.png`

You can use transparent PNG files. The game works if some files are missing, but your own images will look better.
